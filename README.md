# Anidex
A guide to explore and track animes using the Jikan Api. Anidex provides the most popular animes in 6 different categories on the main screen. To provide users with something new, one category at the main display changes everytime app restarts. Users can add the animes to their watchlist, take a peek at the details and search animes that they want to watch.
